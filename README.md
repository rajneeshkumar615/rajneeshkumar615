### Projects

**StayNest** — Property Rental Platform &nbsp; [![GitHub](https://img.shields.io/badge/GitHub-Source-181717?style=flat-square&logo=github)](https://github.com/rajneeshkumar615/StayNest)

> Stack: Node.js · Express · MongoDB · Passport.js · Cloudinary · EJS

Full-stack rental marketplace where hosts manage listings with image uploads and guests leave reviews — secured with session-based auth and role-based route protection. Designed around a clean RESTful API with relational data modeling across User → Listing → Review.

Key engineering decisions: Multer + Cloudinary pipeline for multi-image uploads; connect-flash for stateless UX feedback; authorization middleware preventing cross-user data mutation.
